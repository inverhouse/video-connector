# Video Connector

複数の動画ファイルを並べ替え・結合し、1080p/60fps/MP4で出力するデスクトップアプリ

## 機能

- 🎬 **複数動画の結合** - ドラッグ＆ドロップで順番を変更して結合
- 📱 **iPhone/ハンディカム対応** - MOV, MP4, MTS, M2TS形式をサポート
- 🎯 **統一出力** - 1920x1080 / 60fps / H.264 / AACで出力
- ⚡ **自動変換** - 異なるフォーマットを自動で変換して結合

## 対応フォーマット

| カメラ | 対応形式 |
|---|---|
| iPhone | `.mov`, `.mp4` (H.264/HEVC) |
| Sony ハンディカム | `.mts`, `.m2ts`, `.mp4` (AVCHD) |
