# Release Notes

## v1.0.3 (2026-02-05)

### 🐛 バグ修正

- **音声が出力されない問題を修正**
  - 変換処理で音声ストリームが正しくマッピングされていなかった問題を解決
  - 結合時の音声互換性を改善（異なる形式の動画を結合しても音声が保持されるように）

### 🔧 改善

- 動画変換時の設定を最適化
  - 明示的な映像/音声ストリーム選択 (`-map` オプション)
  - 音声ビットレート: 192kbps
  - サンプルレート: 48kHz (統一)
  - チャンネル数: ステレオ (統一)
  - エンコード品質設定 (`-crf 18`, `-preset medium`)
- アスペクト比を維持するスケーリング処理を改善

---

## v1.0.0 (2026-02-05)

### 🎉 初回リリース

- 複数動画ファイルの結合機能
- ドラッグ＆ドロップで動画の順番を変更
- 1920x1080 / 60fps / H.264 / AAC形式で出力
- 対応フォーマット:
  - iPhone: MOV, MP4 (H.264/HEVC)
  - Sony ハンディカム: MTS, M2TS, MP4 (AVCHD)
- iPhone動画とハンディカム動画が混在している場合の警告表示
